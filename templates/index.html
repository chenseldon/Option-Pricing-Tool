📊 Option Pricing & Risk Analytics Tool

A professional-grade option pricing and risk management tool designed for OTC derivatives roles. Features real-time BSM and Monte Carlo pricing, full Greek calculations, portfolio exposure monitoring, volatility smile analysis, and stress testing — all visualized in an interactive web interface.

Core Features

  • Black-Scholes-Merton pricing with full Greek calculations (Δ, Γ, ν, Θ, ρ)
  • Monte Carlo simulation (up to 200k paths) with P&L distribution histogram
  • Implied volatility solver via Newton-Raphson / Brent method
  • Portfolio exposure aggregation — monitor combined portfolio Greeks
  • MTM valuation & margin estimation per position
  • Volatility smile / skew visualization from market prices
  • Sensitivity stress test matrix (±20% vol, ±60 days)
  • One-click Excel report export (multi-sheet)
